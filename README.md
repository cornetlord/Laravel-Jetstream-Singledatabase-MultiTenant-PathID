## About This Repository

<p> This repostory is everything you need to quickly deploy a laravel 11/Jetstream/Tenancy package. It is designed as a single database tenancy with path identification. </p>

<p>The only changes made to the UI are a place for a company name in the registration form.  Other than that this is a stock Multi Tenant Application </p>
